# Orange Paranoia

Orange Paranoia is a versatile API and NuGet package that provides random information on a wide variety of topics. Whether you need basic random values (like numbers, dates, and strings) or complete/partial data models—with both completely random data and logically related parameters—this project has you covered.

## Features

- **Random Data Generation**  
  Generate basic random values as well as more complex data models.
  
- **Structured & Unstructured Models**  
  Choose between completely random outputs or data models with meaningful relationships between parameters.

- **Dual Interface**  
  - **API:** RESTful endpoints for quick integration into any application.
  - **NuGet Package:** Easily integrate random data generation into your .NET projects.
